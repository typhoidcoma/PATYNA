# Patyna Design System

**Revision:** v1.0\
**Status:** Foundational Brand + UI Spec\
**Owner:** Patyna Product Design

------------------------------------------------------------------------

# 1. Overview

Patyna is a **surface intelligence system**.\
The design language is built around the concept of **topology and
surface fields**.

The brand visualizes data, materials, and structure through **contour
flows** and **clean geometric surfaces**.

Core themes:

-   surface analysis
-   topology
-   material intelligence
-   calm precision
-   engineered minimalism

The visual system should feel closer to:

-   Linear
-   Vercel
-   Stripe
-   Figma

rather than traditional enterprise software.

------------------------------------------------------------------------

# 2. Brand Identity

## 2.1 Name

Patyna

Derived from:

Patina

Meaning:

-   surface character
-   aging of materials
-   accumulated structure over time

This aligns with the product's theme of **surface data and structural
insight**.

------------------------------------------------------------------------

# 3. Logo System

## 3.1 Primary Mark

The Patyna logo consists of a **contour tile symbol**.

Structure:

-   rounded square container
-   flowing contour lines
-   subtle asymmetry forming a P-like flow

Symbol meaning:

-   surface topology
-   data fields
-   material structure
-   computational surfaces

The logo should **not** appear decorative.\
It should feel **engineered and calm**.

------------------------------------------------------------------------

## 3.2 Logo Geometry

Icon container:

-   shape: rounded square
-   corner radius: 20--22%
-   padding: \~18%

Contour lines:

-   count: 4--6
-   stroke weight: \~8--10% of icon width
-   spacing: evenly distributed
-   flow: organic but controlled

Contour flow should subtly form a **P-shaped curvature**.

------------------------------------------------------------------------

## 3.3 Icon Variants

### Primary

-   light background
-   dark contour lines

### Dark Mode

-   dark background
-   white contour lines

### Small Sizes

For icons below **32px**, reduce contour lines to **3 lines maximum**.

------------------------------------------------------------------------

# 4. Wordmark

Primary wordmark:

Patyna

Recommended fonts:

-   Söhne
-   Inter
-   Neue Haas Grotesk

Typography rules:

-   weight: medium
-   tracking: +2 to +4

Avoid:

-   ultra-thin weights
-   decorative fonts

Wordmark should feel **neutral and confident**.

------------------------------------------------------------------------

# 5. Color System

The Patyna palette is inspired by **materials and surfaces**.

Primary colors:

-   Graphite #1C1D21
-   Stone #E7E7E4
-   White #FFFFFF

Accent color:

-   Oxide Blue #4F6F8C

Accent usage should be limited to:

-   active UI elements
-   highlights
-   links
-   selected states

------------------------------------------------------------------------

# 6. Icon System

Icon style rules:

-   stroke-based icons
-   rounded corners
-   minimal detail
-   geometric consistency

Icons should match the **contour tile aesthetic**.

Avoid:

-   heavy fills
-   gradients
-   skeuomorphic effects

------------------------------------------------------------------------

# 7. Motion Language

Motion is inspired by **topographic fields**.

Animations should feel:

-   smooth
-   slow
-   organic
-   computational

Examples:

### Hover

Contour lines shift slightly.

### Loading

Contour ripple expands outward.

### Background

Subtle field movement.

------------------------------------------------------------------------

# 8. Landing Page Design

The landing page should be **minimal and spacious**.

Layout:

-   Logo
-   Hero statement
-   Product preview
-   Feature blocks
-   CTA

Example hero:

Surface intelligence for modern workflows

Backgrounds can include **subtle contour field patterns**.

Opacity: **2--4%**

This creates texture without distraction.

------------------------------------------------------------------------

# 9. UI Design Language

Patyna UI components reflect **surface structure**.

### Panels

-   radius: 10--14px
-   border: #E6E6E6
-   background: white

### Buttons

Primary:

-   bg: Graphite
-   text: white
-   radius: 10px

Secondary:

-   border: graphite
-   bg: transparent

------------------------------------------------------------------------

# 10. Background System

Patyna backgrounds may use **very subtle contour fields**.

Structure:

-   topographic lines
-   very low opacity
-   large spacing

Opacity range: **2--4%**

These backgrounds should feel like **material grain**.

------------------------------------------------------------------------

# 11. App Icon

Structure:

-   rounded square
-   contour field symbol
-   monochrome palette

Icon should be legible at:

-   16px
-   32px
-   64px
-   256px

Reduce contour complexity at smaller sizes.

------------------------------------------------------------------------

# 12. Product Feel

Patyna should feel:

-   calm
-   precise
-   intelligent
-   technical
-   modern

Avoid styles that feel:

-   playful
-   gamified
-   overly decorative

The product should look like **a serious engineering tool with design
sensibility**.

------------------------------------------------------------------------

# 13. Brand Philosophy

Patyna represents the idea that **surfaces contain information**.

Through contour fields and minimal geometry, the brand expresses:

-   structure
-   depth
-   material knowledge
-   computational insight

The goal is a design system that feels **inevitable, quiet, and
intelligent**.
